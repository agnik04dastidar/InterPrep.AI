# InterPrep.AI 🚀

**InterPrep.AI** is an AI-powered mock interview preparation platform that helps users practice realistic interviews, receive intelligent feedback, and improve their communication and technical performance.

The platform simulates real interview scenarios using AI-driven questioning, adaptive follow-ups, and performance analysis.

---

## 🌟 Features

### 🎤 AI Mock Interviews

Practice interviews with an AI interviewer that asks contextual and follow-up questions based on your responses.

### 🧠 Smart Response Analysis

Get detailed insights on:

* Answer quality
* Communication clarity
* Confidence level
* Improvement suggestions

### 🎯 Role-Based Interviews

Prepare for different interview types such as:

* HR Interviews
* Technical Interviews
* Custom Role Interviews

### 📊 Interview Performance Reports

Track performance across sessions and identify areas for improvement.

### 📂 Resume & File Upload

Upload resumes or supporting documents for personalized interview analysis.

### 🔐 Secure Authentication

User authentication system with login, signup, and session management.

### 🎨 Responsive UI

Modern and responsive interface built for seamless use across devices.

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/agnik04dastidar/InterPrep.AI.git
```

### 2️⃣ Navigate to the project

```
cd InterPrep.AI
```

### 3️⃣ Install dependencies

Frontend:

```
cd client
npm install
```

Backend:

```
cd ../server
npm install
```

### 4️⃣ Run the backend server

```
npm start
```

### 5️⃣ Run the frontend

```
cd ../client
npm run dev
```
